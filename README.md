# CU-expand-russification
## Вкратце о руссификаторе
Неофициальный перевод Casualties: Unknown от CheBuRek8176. В тестировании участвовали Roketuss, PROPELLER22869 и Bulaga.
Данный руссификатор был сделан на базе другого руссификатора с официального репозитория, но в итоге был почти полностью переписан из-за корявости перевода. Ссылку на оригинал, увы, я где-то потерял, и вообще я нашёл его, внимание, в Pull Requests.
Идею "расширения" взял из очень древнего руссификатора аш для demo v1, увы, ссылки нет. Его всё ещё можно найти среди первых постов на официальном сайте игры на itch.io. А идею советов взял из Project Silverfish.
**РУССИФИКАТОР РАБОТАЕТ ТОЛЬКО НА ВЕРСИИ ИГРЫ 6.0**
На других версиях будут многочисленные баги.
**В настоящий момент руссификатор находится в разработке, и в нём много опечаток и отсутствующих переводов. И ты можешь помочь ему! Предлагай предложения по переводу в Pull Requests**
## Установка

 - Скачай JSON-файл (пока что я не особо разбираюсь в этом, так что тыкаешь на файл в репозитории, затем на три точки и затем на Download, либо на файл, затем на Download raw file)
 - Закинь файл по пути: C:\Program Files (x86)\Steam\steamapps\common\Casualties Unknown Demo\CasualtiesUnknownDemo\CasualtiesUnknown_Data\Lang (если у тебя игра загружена на диск C:)
 - Зайди в игру и нажми на кнопку переключения языка в меню (справа сверху)
 - Всё!
